Paybox as a WordPress's GiveWP payment gateway.

== Description ==

**[Give](https://givewp.com)** is a donation solution for WordPress.
**[Paybox](http://www1.paybox.com/)** is an online payment gateway.

This plugin adds the ability to make payments on a Give-enabled WordPress using the Paybox (CB/visa) payment system.
